**Introduccion:** 

- Los archivos HTML estan disenados para ejecutarse en navegadores, asi como los videos tiene reproductores de videos para reproducirlos, los archivos de audio tiene reproductores de audio. Los archivos de texto (donde se encuentra el codigo de HTML y CSS dentro) tiene Navegadores WEB que los reconocen, reproducen y muestran al usuario. Estructurados de forma tal que el navegador, muestre lo que el desarrolador quiera que muestre.

- Ej. *<p> este es un texto </p>* : esto muestra un parrafo simple
<*button> boton </button>* : esto muestra un boton 
<img src="perrito.png"> : esto muestra una imagen llamada perrito.png 

- Los navegadores estan hechos especificamente para reproducir estos codigos.

**Parte logica:**

    Que es HTML:
        Significa: 
        Hypertext Markup Lengauge(o lenguaje de marcado de Hipertexto):

            -Lenguaje de marcado: es un sistema de texto que necesita un programada para poder comprender que quiere decir con ese sistema, hay lenguajes de marcado dificiles como HTML o sumamente faciles como es Markdow(lenguaje de marcado ligero). 

                -Markdown si escribimos un texto simple, lo puede interpretar un procesador de Markdown por ejemplo si escribimos: *texto* markdown lo interpreta como: Texto(en negrita), por lo que podriamos usarlo para restaltar palabras de un texto. Si escribis: #texto ~ se va a interpretar como un encabezado grande.

            En cambio en HTML si queremos resaltar un texto en negrita lo hacemos de la siguiente forma: <b>texto</b> , si queremos un ecabezado grande lo hacemos de esta otra: <h1>Texto</h1>.

            Este es el objetivo de los lenguajes de marcado, que nosotros escribamos texto y esto sea interpretado por un programa que entienda lo que queramos decir y en que formato.

            Primer caracteristica de HTML: es INTERPRETADO

    Para que sirve HTML:
        Con HTML lo que hacemos es armar la ESTRUCTURA (literalmente) de la pagina web. Cada ELEMENTO es o un BLOQUE o algo que le da info extra al BLOQUE. Los ELEMENTOS son cada texto que coloquemos en el codigo, por ej.
            <!DOCTYPE html>
            <html>
            <head>
                <title>Mi Sitio Web</title>
            </head>
            <body>
            <!-- Seccion de Navegacion>
            <div>
                <a href="inicio.html">Enlace 1</a>
                <a href="noticias.html">Enlace 2</a>
                <a href="deportes.html">Enlace 3</a>
                <a href="perfil">enlace 4</a>
            </div>
        Esto genera 4 botones de indice de navegacion para mostra al usuario. Con HTML creamos los elemntos que vamos a mostrar en pantalla, y la estructura de estos. Estos elementos se llaman: Tags o Etiquetas por que literal lo que hacen es eso, etiquetar el contenido.
            
            Ej. Si yo escribo <b> </b> todo lo que escriba adentro, se va a mostrar en negrita.
            
            Propriedades: 
                ej. Tamano de texto, color de texto, color de fondo, sombra, etc. Esto es definido por el navegador. el texto por defecto queda de color negro, el tamano del texto base por defecto suele ser 16px y asi.
        
    CSS: 
        CSS vino para dar CARACTERISTICAS al codigo, para que TODOS los navegadores, reconozcan las propiedades que el desarrollador decide.

        Ej. CSS funciona en base a reglas de estilo. Se puede definir:

            .CUBO {   
                COLOR: AZUL;
                
                TAMANO: 16PX;

                MARGEN: 20PX;

                BORDE: NEGRO;
                }

        la estructura de una regla de estilo de CSS consta de: Un selector y un Bloque de declaraciones.

            Selector: 
                .CUBO{ 

                    hace referencia al elemento de HTML que le queres cambian las propiedades.
            
            Bloque de delcaraciones:
                
                COLOR: AZUL;
                
                TAMANO: 16PX;

                MARGEN: 20PX;

                BORDE: NEGRO;
                }

                Tiene todas las propiedades que queremos modificar y cual va a ser el valor que queremos que tenga.

PARRAFOS y ENCABEZADOS:
    Toda pagina web tiene una estructura base, en VSCode, escribiendo: html:5, autocompleta el codigo con la siguiente estructura:

        <!DOCTYPE html>
    <html lang="en">
    <head>
        <title>Document</title>
    </head>
    <body>
    
    </body>
    </html>

<html></html> : es la etiqueta que le dice al navegador, que todo lo que este dentro, es nuestra pagina web.

<head></head> : le da informacion al navegador al respecto de nuestra web, lo que no podemos ver.
            ej. <title></title> es la etiqueta que se utilizar para darle nombre a la pagina (el texto que aparece en la pesatana de navegacion)

<body></body> : dentro del body, va a estar toda la web VISIBLE, toda la pagina web, al construimos dentro de body.

<p></p> son PARRAFOS donde vamos a dejar contenido. Cada parrafo que escribimos, tiene que tener

<h1></h1> el encabezado, por pagina web se utiliza solo UN h1. los h2 son subtitulos. Los h3, h4, h5 y h6 estan disenados para los mini subtitulos
        que podemos utilizar dentro de un subtitulo. mientras mayor el nuemro del H* mas chico el tamado del subtitulo.

ATRIBUTOS: Son propiedades de los ELEMENTOS de html
            href=""  es el atributo utilizado para referir enlaces.

            target="" es el atributo que se utiliza para elegir a donde se va a abrir el ENLACE que clickiemos. 
                predeterminado viene target="_self" se abre en la misma pagina actualizandola
                pero podemos utilizar target="_blank" para que el enlace se abra en una nueva pestana.

            title="" es utilizado para darle un titulo en forma de informacion al respecto de el elemento donde coloquemos el atributo. Permitiendo dar informacion extra al usuario que este utilizando la web.

            alt="" es el atributo para colorar un texto en caso de que una imagen no cargue. Y describe completamenta la imagen para posicionamiento de google.

            ../  es el atributo que se utiliza para volver atras o buscar un archivo en una carpeta anterior.

            type="" sirve para elegir que tipo de caja de interaccion vamos a utilizar en la web, ej. type="text"(caja de texto), type="checkbox"(checkbox de toda la vida), y asi varios.

            name="" Es el identificador natural de cada input EN HTML

            required es para indicar que el campo tiene que completarse si o si para enviar el formulario

            placesholder="" deja un pequeno mensaje que desaparece cuando el usuario comienza a escribir, dentro de la caja de texto.

CSS : cascading style sheets o en espanol Hojas de Estilo en Cascada.
        CSS lo utilizamos para agarrar componentes de HTML y darle un ESTILO.

    Lo primero es diferenciar ATRIBUTOS de PROPIEDADES. Entendiendo que atributos viene de HTML (ej. href o title). Y las propiedades de CSS teniendo una forma diferente de acceder a estas.
    
    Tenemos 3 formas de ENLAZAR CSS.

        La PRIMER forma es algo que se llama: Estilos en LINEA, por que normalmente se da estilo a una sola linea de codigo. (es una mala practica)
        
        la SEGUNDA forma es utilizando STYLE pero no como atributo sino como ETIQUETA
        <style>
            p {
             color:blueviolet
            }
            h1 {
                color:blue
            }
        </style>
        Aun que TAMPOCO se recomienda hacer esto.

        La TERCER forma, es una etiqueta especial que se llama LINK, que justamente LINKEA una hoja de estilo. Esta hoja de estilo se suele poner en el HEAD. 
        dentro de la etiqueta <HEAD> vamos a colocar lo siguiente:

        <link rel="stylesheet" href="styles.css">

            rel: haciendo referencia a RELATIONSHIP

            "stylesheet": haciendo referencia a una hoja de estilo

            href: indicando el hipervinculo de donde va a provenir ese estilo que vamos a darle a la pagina.

    una rapida explicacion de como se organiza la hoja de estilo seria:

            p {
                color: cadetblue;
                font-size: 30px;
            }
            h1{
                color:chocolate;
                font-size: 40px;
            }

    Entendiendo que "p" o "h1" son los ELEMENTOS a los cuales darle formato.

    Los {} para encapsular el estilo que vamos a darle a esos elementos. BLOQUE DE DECLARACIONES.

CSS Selectores: Son la forma de seleccionar uno o mas elementos en HTML. Pero esta vez es CSS para poder modificarlos.

        p {
            color: cadetblue;
            font-size: 30px;
        }

        Esta es la forma de seleccionar todos los elementos que se llamen (p) y modificarlo.

    CLASS: es el atributo que se utiliza para deparar las cosas por CLASE, por ejemplo, quiero que este elemento sea una caja giratoria y le asignamos el elemento "class="caja giratoria"". Cuando queramos modificar todos los elementos lo modificamos una vez y es como un PRE-SET , una plantilla. 
    En la parte de CSS se escribe: .class 
            ej.

            .faltante{
                        color:red
            }

    ID: es otra forma de clasificar elementos, utilizando el selector ID. Es un selector que se le puede dar a UN elemento en toda la web.
    En la parte de CSS se escribe: #ID
            ej.

            #ingrediente-secreto{
                                    color:rgb(15, 139, 139)
            }



PROPIEDADES DE FUENTE Y TEXTO:

    font-family: nos permite elegir la tipogracia.

    font-size: cambia el tamano de la tipografia.

    font-style: tiene 3 importantes. ITALIC, que pone la letra en cursiva o media de costado. NORMAL, valor por defecto. OBLIQUE, podemos seleccionar cuantos grados pueden estar inclinada la letra.

    text-align: es la propiedad que decide la ubicacion del texto, izquierda, derecha, centrada, justificada, etc.
        lo que hace es alinear el texto en base al contenido, en h1 al ser un titulo de bloque, puede agrupar todo contenido y centrarlo, lo mismo con los <p> toma todo el texto del bloque <p> y lo modifica.
    
    text-decoration: tiene varias opciones, como subrayar por debajo, o el subrayado por encima. O tachar todo el texto.

    line-heigth: es para seleccionar el espaciado entre texto.

    letter-spacing: Es para el espaciado entre letras.

    text-transform: puede convertir todo el texto en mayusuculas, o todo el texto en minusculas. CAPITALIZE: 



TIPOGRAFIAS EXTERNAS:
    un pequeno tutorial de como agregar tipografias externas, creamos un nuevo <LINK> en el index, una vez descargadas las tipografias y colocadas en la carpeta ASSETS, podemos cargar desde fonts.css las tipografias nuevas y darle los parametros correspondientes.
        ej.
            @font-face {
            font-family: "monserrat";
            src: url("fonts/Montserrat-Regular.ttf") format("truetype");
            font-weight: 400;
            }

            @font-face {
            font-family:"monserratblack";
            src: url("fonts/Montserrat-ExtraBold.ttf") format("truetype");
            font-weight: 700;
            }

        Asi es la forma de cargar la tipografia nueva, externa.


BOX MODEL!:

    todo funciona con CAJAS, las cuales tiene CUATRO propiedades principales.

    box model es un concepto, de, como se disenan y como se deberian colocar los elementos en HTML

    cada elemento en HTML es una CAJA, esta caja tiene PROPIEDADES que las definen.

        CONTENT: es el nucleo del boxmodel, basicamente es el area donde se muestra el texto, img, video, etc.

        PADDING: es el relleno, o el espacio entre el contenido y el borde del elemento. Este espacio aumenta el area al rededor del contenidod el elemento pero debajo el borde.

        BORDER: es una linea que rodea tanto el contenido, como el padding, y se puede modificar el estilo, el grosor y el color.

        MARGIN: es el pesacio alrededor de la caja que esta fuera del borde. Este espacio separa el elemento, de los otros elementos de la pagina.
